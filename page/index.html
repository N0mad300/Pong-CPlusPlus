Pong C++

About

It's my first C++ project with SDL2, the main goal of this project for me was to discover SDL2 and SDL TTF.

This project help me also to familiarize me with the Oriented Object Programmation in C++

Gameplay

In this recreation of Pong you can control the left paddle using W and S and play against a very basic AI.

Controls

W/S : Up/Down (Left paddle)

R : Reset the game

P : Pause the game
